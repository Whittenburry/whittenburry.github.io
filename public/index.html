Experience

Voltz Software

  • I spent the last year working as a contractor for Voltz Software assisting in the development of a business software application called Bento.Systems. While working there, I was exposed to a few new concepts regarding functional programming and the overall essence of JavaScript and its architecture. A good example of this would be Nicholas Zakas and his thoughts on Module Theory regarding Scalable JavaScript Application Architecture. My responsibilities were equally as exciting. Using an in-house, proprietary framework I assisted in the design, implementation and maintenance of numerous new and pre-existing features being added or updated within the application we were developing. A few specific features I was solely responsible for were the Commenting System and the Timeline/Activity Feed.

    The commenting system was designed to have all the core functionality you would expect to see when using any kind of social media chat feature (i.e. inline @userName mentioning, object tagging, etc). In addition to the baseline behavior, I created individual feature sets for each sub system that comments would be integrated with.

    The Timeline/Activity Feed was modeled after Basecamp's Activity Feed. By visually depicting any & all activity within a certain team, users are capable of maintaining all the relevant data pertaining to any system in one seamless flow chart from start to finish. Subsequently gaining the ability to interact with it in one unified space, without forcibly being made to navigate away from the view.

    I also made contributions to the framework itself; occasionally adding snippets of reusable code to ensure its functionality continued to meet our needs and coincide with our growth. All in all, it was a great place to work and I'm excited to see where the company will be 10 years down the road.
